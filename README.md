The Course Enrollment is a Java-based console application developed using JDBC and Object-Oriented Programming concepts.  
It allows students to enroll in courses, manage enrollments, and handle course capacity with proper validations and exception handling.

The project follows a layered architecture with:

- DAO Layer  
- Service Layer  
- Utility Layer  
- Entity Layer  

 Technologies Used

- **Programming Language:** Java  
- **Database Connectivity:** JDBC  
- **Database:** Oracle / MySQL (Configurable)  
- **IDE:** Eclipse  
- **JDK Version:** Java SE 22
